Landing Page Project

This is a beginner-friendly landing page built with simple HTML & CSS. It demonstrates layout techniques (flexbox, grid), responsive breakpoints, and easy customization via CSS variables.

Files
- index.html — main page
- css/styles.css — styles

How to view
1. Open `c:\Users\user\OneDrive\Desktop\oasis intern\login page\landing-page\index.html` in your browser (double-click or right-click -> Open with).
2. Resize the browser to see responsive behavior.

Next steps you can try
- Tweak colors in `css/styles.css` variables.
- Replace the SVG with an image or your own illustration.
- Add Google Fonts by adding a link in `index.html` head.
